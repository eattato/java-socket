# 자바 소켓 프로그래밍 - 채팅 프로그램
## 개발 동기
* 메신저 프로그램인 카카오톡과 디스코드의 동작 원리에 대한 탐구
* Java 및 Java 소켓 프로그램의 이해
* 서버, 클라이언트의 네트워크 연결에 대한 이해도 증진
* Thread 이해도 증진

## 구성
서버 (socket)
클라이언트 (socket-cl)

## 기능
1. 서버, 클라이언트 접속
2. 채팅방 참가 및 생성
3. 채팅방 생성 시 채팅방 설정 변경
3-1. 채팅방 모드 변경
3-2. 익명 모드
3-3. 파일 전송 허용 여부
4. 메시지 전송
5. 이미지 파일 전송
6. 닉네임 변경
7. 프로필 사진 변경
